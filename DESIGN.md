# DESIGN.md - 濒危生物趣味交互科普

## 气质与意象

产品核心意象：**「探险者的秘境手记」**——你是一位踏入无人之境的探险者，随身携带着一本古老的精灵图鉴。在迷雾中触碰微光，翻开图鉴记录每一次相遇，每一页都承载着一个濒危生灵的故事。

- **森林场景**：黄昏时分的原始密林，最后一缕天光穿透树冠化为金色光柱，萤火虫开始起舞。苔藓覆盖的古树间雾气蜿蜒，你站在一处被遗忘的神圣林地，珍稀动物以发光精灵的形态显现。
- **海洋场景**：漂浮在深渊海沟中，生物荧光如水下星河缓缓飘移。水中微粒捕捉最微弱的光，压力与寂静压缩为冥想空间。深海的异域之美，每个气泡里都封存着一个秘密。

## 精灵图鉴视觉系统

### 核心视觉语言
- **旧纸张质感**：泛黄奶白底色 `#F5ECD7`，搭配微妙噪点纹理和纸张纤维感
- **手绘装饰线**：墨色描边、虚线标注、手写体花饰，营造田野笔记的真实感
- **复古标签贴纸**：场景名/提示用旧标签贴纸风格，带折角和半透明胶带效果
- **皮革装帧**：卡片边框使用暗棕皮革色 `#3D2B1F`，搭配压印纹理感
- **金属铜扣**：卡片角落、书脊处使用铜色 `#B87333` 装饰扣

### 字体排版

- **字体族**：`'PingFang SC', 'Noto Sans SC', system-ui, sans-serif`
- **图鉴标题**：20-24px，font-weight 600，letter-spacing 0.05em，墨色
- **图鉴正文**：13-15px，font-weight 400，line-height 1.8，淡墨色
- **手写标注**：12px，font-style italic，letter-spacing 0.03em，模拟手写感
- **标签贴纸**：11-12px，font-weight 500，大写追踪 `tracking-wider`
- **编号戳章**：10px，font-weight 700，等宽数字

### 纸张与纹理

- 纸张用CSS渐变+噪点模拟：`radial-gradient(ellipse, #F5ECD7, #E8DCC6)` + 微弱noise overlay
- 卡片边缘有做旧效果：box-shadow内发光 + 微弱边框破损感
- 装饰线用虚线：`border: 1px dashed rgba(44,24,16,0.2)`
- 角落有铜扣装饰：圆形8px，铜色渐变+阴影

### 信息卡片（物种图鉴页）

- 整体：旧纸底色，四角铜扣装饰，顶部皮革色装帧线
- 左页/右侧布局：上方物种图，下方档案区
- 档案区用手绘虚线分割各字段，字段名用淡墨小字标注
- 保护等级用印章样式：圆形红印，等级文字居中
- 拉丁名用手写体样式，斜体+淡墨色
- 趣味知识用"批注"样式：左侧竖线 + 略缩进 + 斜体
- 底部有发现编号戳章：`No. 012 / 138`

### 欢迎页

- 整体：翻开的笔记本双页展开效果
- 左页：手绘风格的世界地图轮廓（简约线条），标注探险路线
- 右页：手写体欢迎文字，墨色渐淡效果
- 纸张有做旧的茶渍、折痕
- 「踏入秘境」按钮用复古蜡封印章风格
- 环绕粒子改为飘散的金色纸屑/墨点

### 场景UI元素

- **场景名**：复古标签贴纸，带折角 + 半透明胶带条，贴在画面左下
- **场景描述**：标签下方手写批注风格
- **物种计数**：右上角批注样式，手写编号 `已记录 3 / 138`
- **罗盘导航**：复古黄铜指南针造型，中心指针缓慢旋转

### 动效规范

- 卡片翻开：从右向左翻开书页效果，带纸张弯曲阴影
- 标签贴纸：出现时微微摇摆一下再贴稳
- 计数更新：数字用手写覆盖效果（旧数字划掉+新数字写上）
- 罗盘指针：缓动旋转，有金属质感的光泽变化

## 设计禁忌

- **禁止**扁平纯色背景——场景必须有多层渐变、雾气、光效
- **禁止**直角锐边——所有容器至少 `border-radius: 8px`
- **禁止**高饱和霓虹色——所有色彩饱和度不超过 50%
- **禁止**纯现代UI组件——所有界面元素必须融入图鉴/笔记风格
- **禁止**纯白/纯黑——纸色用奶白，墨色用深棕
- **禁止**粗体大字破坏沉浸感——文字是信息辅助，不是主角
