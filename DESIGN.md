# DESIGN.md - 濒危生物趣味交互科普

## 气质与意象

产品核心意象：**「秘境探索」**——用户是一位踏入无人之境的探险者，在迷雾中用手指触碰微光，揭开自然深处隐藏的珍稀生灵。

- **森林场景**：黄昏时分的原始密林，最后一缕天光穿透树冠化为金色光柱，萤火虫开始起舞。苔藓覆盖的古树间雾气蜿蜒，你站在一处被遗忘的神圣林地，珍稀动物以发光精灵的形态显现。
- **海洋场景**：漂浮在深渊海沟中，生物荧光如水下星河缓缓飘移。水中微粒捕捉最微弱的光，压力与寂静压缩为冥想空间。深海的异域之美，每个气泡里都封存着一个秘密。

## 字体排版

- **字体族**：`'PingFang SC', 'Noto Sans SC', system-ui, sans-serif`——纤细圆润无衬线体
- **标题**：18-22px，font-weight 500，letter-spacing 0.02em
- **正文**：13-15px，font-weight 300-400，line-height 1.7
- **标签/徽章**：11-12px，font-weight 400，大写追踪 `tracking-wider`

## 视觉策略

### 磨砂玻璃与半透明
- 所有覆盖层、卡片、弹窗使用 `backdrop-blur(16px)` + 半透明白色/深色底
- 森林：`rgba(30,40,28,0.75)` 底 + 模糊
- 海洋：`rgba(10,22,40,0.75)` 底 + 模糊
- 边框：`1px solid rgba(255,255,255,0.08)`

### 信息卡片
- 圆角 `border-radius: 20px`（柔边，接近椭圆感）
- 无硬阴影，使用 `box-shadow: 0 0 40px rgba(color,0.15)` 柔光扩散
- 内边距充裕，留白即呼吸

## 动效规范

### 载体运动（光斑/气泡）
- 漂浮：`cubic-bezier(0.25,0.46,0.45,0.94)`，周期 4-8s
- 呼吸缩放：`scale(1)→scale(1.15)→scale(1)`，周期 3-5s
- 微偏移：`translateX/Y` ±5-15px，周期 6-10s

### 点击反馈
- 光斑被点击：缩放弹跳 `scale(1)→1.3→0` + 光晕爆发扩散，300ms
- 气泡被戳破：`scale(1)→1.15→0` + 碎片飞散，400ms
- 解锁闪光：径向渐变从中心扩散，500ms

### 弹窗出场
- 卡片从底部滑入 + 淡入：`translateY(30px)→0` + `opacity 0→1`，400ms，`cubic-bezier(0.16,1,0.3,1)`
- 退场：反向，300ms

### 场景切换
- 交叉淡入淡出：`opacity 1→0` / `0→1`，800ms
- 缩放过渡：当前场景 `scale(1)→1.05` + 淡出，新场景 `scale(0.95)→1` + 淡入

## 页面结构

### 全屏沉浸式场景
- 无导航栏，场景占满视口
- 左下角：场景名称 + 简短描述（半透明文字）
- 右下角：场景切换按钮（胶囊形态）
- 顶部居中：已解锁物种计数 `3/8`
- 点击载体后：居中弹出物种卡片，遮罩层模糊背景

## 设计禁忌

- **禁止**扁平纯色背景——必须有多层渐变、雾气、光效
- **禁止**直角锐边——所有容器至少 `border-radius: 12px`
- **禁止**高饱和霓虹色——所有色彩饱和度不超过 50%
- **禁止**机械线性动画——所有动效使用缓动曲线
- **禁止**传统「按钮/卡片」外观——交互载体是自然形态（光斑/气泡）
- **禁止**浅色/白色背景——双场景均为暗色大气风格
- **禁止**粗体大字破坏沉浸感——文字是信息辅助，不是主角
